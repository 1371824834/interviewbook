# 关键字

\[struts\]``

# 说出struts中的标签。【基础】

# 参考解答：



```

      
<bean:message />       <html:errors />        <logic:present />
<bean:define />        <html:messages />      <logic:equal />
<bean:include />       <html:html>
<bean:write />         <html:form>
<bean:resource />      <html:link>
<bean:cookie />        <html:text>
<bean:heder />         
<bean:size />          
<bean:parameter /> 

```

详细解释：



```
<bean:message />：用于显示资源文件中的消息文本，多用于国际化               
<bean:define />：用于定义一个变量              
<bean:include />：将其他web资源包含进当前页面中      
<bean:write />：用于输出各种类型的对象，类似于System.out.println()         
<bean:resource />：用于检索、获得web资源内容，如网页源代码      
<bean:cookie />:访问cookie信息        
<bean:heder />：访问HTTP请求中的Header信息         
<bean:size />：该标签用于获取结合或者数组的长度          
<bean:parameter />：访问请求参数 
<html:errors />：用于输出错误消息 
<html:messages />：用于输出消息
<html:html>：用于在网页开头生成HTML的<html>元素
<html:form>：用来创建表单，必须包含一个action属性，否则JSP会抛出异常
<html:link>：用来传递参数
<html:text>：用于创建表单上的文本框字段，其属性指定了该文本框的名字
<logic:present />:判断指定的对象是否为空
<logic:equal />：进行类似于if判断
```
