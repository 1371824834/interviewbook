# 当一个对象被当作参数传递到一个方法后，此方法可改变这个对象的属性，并可返回变化后的结果，那么这里到底是值传递还是引用传递? 

# 参考解答

java 中只有值传递，看下面两个例子：

```java
public static void main(String[] args) {
  Cat c1 = new Cat();
  c1.name = "小黑";
  foo(c1);
  System.out.println(c1.name);
}

public static void foo(Cat c2) {
  c2 = new Cat();
  c2.name = "小白";
}
```
