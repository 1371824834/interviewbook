#　写一个函数，要求输入一个字符串和一个字符长度，对该字符串进行分隔。

# 参考解答

```
public String[] split(String str, int chars){
    int n = (str.length()+ chars - 1)/chars;
    String ret[] = new String[n];
    for(int i=0; i<n; i++){
        if(i < n-1){
            ret[i] = str.substring(i*chars , (i+1)*chars);
        } else{
            ret[i] = str.substring(i*chars);
        }
    }
    return ret;
}
```
