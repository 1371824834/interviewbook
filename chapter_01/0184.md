# 写出下面代码的输出结果。

```java
class A {
	public static void hello() {
		System.out.println("A");
	}

	private void showMsg() {
		System.out.println("This is A");
	}

	public void show() {
		showMsg();
	}
}

public class B extends A {
	public static void hello() {
		System.out.println("B");
	}

	private void showMsg() {
		System.out.println("This is B");
	}

	public static void main(String[] args) {
		A a = new B();
		a.show();
		a.hello();
	}
}
```

# 参考答案

This is A
A
