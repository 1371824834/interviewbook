# 页面中有一个命名为Bank No的下拉列表，写脚本获取当前选项的索引值。
# 参考答案
#### 

```html
<select name="Bank No" >
  <option value="1">中国银行</option>
  <option value="2">招商银行</option>
  <option value="3">建设银行</option>
  <option value="4">农业银行</option>
</select>
```

当选中`建设银行`时，执行：
```js
document.getElementsByName("Bank No")[0].value;
// 输出 3
```
